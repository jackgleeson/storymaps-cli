## How to Contribute

### Bug Reports

Open an issue on [GitHub](https://github.com/jackgleeson/storymaps.io/issues) with:
- A clear description of the problem
- Steps to reproduce
- Expected vs actual behaviour
- Browser and OS details if relevant

### Pull Requests

1. Fork the repository
2. Create a feature branch from `main`
3. Make your changes
4. Test locally
5. Submit a pull request with a clear description of the change

By opening a pull request, you confirm that you have read and agree to the Contributor Licence Agreement above.
